
1. 서버: 사용자가 요청하면 서비스를 제공하는 프로그램

2. 웹서버(http서버): 사용자가 요청하면 html을 보내주는 서버 프로그램 ex) 우리가 아는 모든 웹 사이트는 http서버가 돌린다
   cf1) http는 웹에서 사용하는 통신표준
   cf2) https는 http + secret. 즉 보안이 적용된 http

3. http요청을 보내려면 서버의 IP주소와 포트를 알아야 한다

4. 사용자는 203.253.146.184와 같은 IP를 기억하기 어렵다.
   => 그래서 www.naver.com과 같은 도메인이름(domain name)을 사용한다

5. DNS: 도메인 이름을 IP주소로 바꿔주는 서비스
   사용자가 도메인 이름을 입력하면 DNS가 IP로 변환한다
